# Weather-Prediction-ML-Project
📌 Project Overview

This project uses historical weather data to predict whether it will rain the next day. By analyzing factors such as temperature, humidity, rainfall, wind speed, and pressure, the model helps make accurate weather predictions using Machine Learning techniques.

✨ Features
Weather data preprocessing and cleaning
Missing value handling
Outlier detection and treatment
Feature engineering from date columns
Categorical variable encoding
Data visualization and EDA
Rain prediction using classification models
🛠️ Tech Stack
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
Category Encoders
🤖 Models Used
Logistic Regression
Decision Tree Classifier
Random Forest Classifier
📈 Key Results
Successfully trained models to predict rainfall for the next day
Improved dataset quality through preprocessing and feature engineering
Achieved reliable classification performance using ensemble methods like Random Forest
Generated insights from weather trends through visual analysis
