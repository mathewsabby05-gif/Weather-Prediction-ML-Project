# 🌦️ Weather Prediction ML Project

## 📌 Project Overview
This project uses historical weather data to predict whether it will rain the next day. By analyzing factors such as temperature, humidity, rainfall, wind speed, and pressure, machine learning models are used to make accurate weather predictions.

## ✨ Features
- Weather data preprocessing and cleaning  
- Missing value handling  
- Outlier detection and treatment  
- Feature engineering from date columns  
- Categorical variable encoding  
- Exploratory Data Analysis (EDA)  
- Rain prediction using classification models  

## 🛠️ Tech Stack
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  
- Category Encoders  

## 🤖 Models Used
- Logistic Regression  
- Decision Tree Classifier  
- Random Forest Classifier  

## 📈 Key Results
- Successfully trained models to predict rainfall for the next day  
- Improved dataset quality through preprocessing and feature engineering  
- Achieved reliable prediction performance using Random Forest  
- Generated useful insights from weather trends through data visualization
